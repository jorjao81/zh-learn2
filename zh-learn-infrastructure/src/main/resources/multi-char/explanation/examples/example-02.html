Origin: Compound word (复合词)

Modern compound 电脑diànnǎo combines: 电 (electricity) + 脑 (brain): the first character 电diàn refers to electrical power, while the second character 脑nǎo means brain or mind, suggesting intelligence.

This is a 20th century neologism created to describe computers by combining the concepts of electrical operation and brain-like processing capabilities.

The Semantic Evolution:

"electricity + brain" → "computer." The compound meaning creates a metaphor for an electronic thinking machine.

Became the standard Chinese term for computers. Examples: 台式电脑táishì diànnǎo "desktop computer"; 笔记本电脑bǐjìběn diànnǎo "laptop computer."

Path: 电diàn "electricity" + 脑nǎo "brain" → "electric brain" → "computer; electronic processing device."
