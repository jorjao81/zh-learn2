**确** (què): authenticated / solid / firm / real / true

- **Etymology**: Stone (石) + High/Corner bird (u/c). Meaning evolved from "hard like stone" to "firm/certain."
- **Associations**: 确定 (determine/certain), 正确 (correct), 准确 (accurate). All imply hitting a specific, hard target of truth.

**实** (shí): real / true / solid

- **Meaning**: Substance, reality.
- **Combination**: **确实** combines "firm/certain" + "reality." It is a "solidified reality."
